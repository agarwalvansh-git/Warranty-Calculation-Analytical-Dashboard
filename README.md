📊 Warranty Analytical Dashboard

A data-driven dashboard designed to automate and enhance warranty planning using historical sales and usage trends. This tool provides key insights into inventory, build requirements, and projected demand before End-of-Support (EOS), enabling smarter and faster decision-making.

🚀 Key Features
🔹 Automated Warranty Planning
Uses historical sales and usage data to estimate future warranty requirements.
🔹 KPI Tracking
Monitors critical metrics such as:
Stock on Hand
Estimated Build Requirements
Projected Sales before EOS
🔹 Demand Forecasting (Pre-EOS)
Helps anticipate product demand before end-of-support timelines.
🔹 Inventory Visibility
Provides a clear view of available stock vs expected needs.
🔹 Reduced Manual Effort
Eliminates manual calculations and improves efficiency through automation.

📈 Dashboard Overview
The dashboard includes:

Stock on Hand – Current inventory availability
Estimated Build Needs – Required FG units for warranty support
Projected Sales (Pre-EOS) – Expected demand before product lifecycle ends
Trend Analysis – Historical sales and usage patterns
Interactive Visuals – Easy-to-understand charts and KPIs
🛠️ Tech Stack
Power BI (Dashboard & Visualization)
Data Modeling & Transformation (ETL)
Excel / Database Sources (based on input data)

📂 Use Cases

This dashboard is ideal for:

Supply Chain & Planning Teams
Warranty & After-Sales Support Teams
Inventory Management
Business Strategy & Forecasting
